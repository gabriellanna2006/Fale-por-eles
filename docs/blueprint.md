# **App Name**: Fale Pelos Animais

## Core Features:

- Incident Reporting Form: Users can submit detailed reports of animal abuse incidents, including descriptions, locations, and media uploads.
- Report Mapping: Display reported incidents on a map of Raul Soares, providing a visual representation of animal abuse hotspots.
- Animal Welfare Resources: A curated directory of local animal shelters, veterinary services, and animal welfare organizations in Raul Soares.
- Educational Content: Informative articles and resources on animal rights, responsible pet ownership, and preventing animal cruelty.
- Image Analyzer Tool: A tool which receives uploaded images of possible animal cruelty as input. The tool uses the image and other user input to estimate the degree to which the animal may be subject to harm, abuse or neglect; provide tailored informational resources that correspond to the tool's estimates of abuse or harm.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to convey trust and compassion.
- Background color: Light grey (#F5F5F5) for a clean, neutral backdrop.
- Accent color: Gentle green (#81C784) to symbolize growth, health, and hope.
- Body and headline font: 'PT Sans', a humanist sans-serif for readability and a modern feel.
- Use simple, easily recognizable icons to represent report categories and resources.
- Clean and responsive design, ensuring accessibility on all devices.
- Subtle transitions and animations to guide user interactions.
